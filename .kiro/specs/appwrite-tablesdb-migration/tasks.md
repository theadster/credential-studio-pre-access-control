# Implementation Plan: Appwrite TablesDB Migration

## Overview

Migrate credential.studio from the legacy Appwrite `Databases` API to the new `TablesDB` API. The migration follows a bottom-up approach: environment variables ‚Üí client factory ‚Üí library modules ‚Üí API routes ‚Üí tests ‚Üí scripts ‚Üí docs.
